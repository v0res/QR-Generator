#  Batch QR-Code Generator

Инструмент на Python для автоматизированной генерации QR-кодов. Поддерживает как одиночную работу, так и **пакетную генерацию** из текстового файла.

## Основные возможности
* **Пакетная обработка**: Автоматическое чтение списка ссылок из `links.txt`.
* **Валидация имен**: Интеллектуальная очистка названий файлов (удаление протоколов, поддержка кириллицы).
* **Модульная архитектура**: Полное разделение логики (Core) и интерфейса (CLI).
* **Контейнеризация**: Готовый Docker-образ для запуска в изолированной среде.
* **Качество кода**: Покрытие тестами (`pytest`) и проверка линтерами (`ruff`, `black`).

## Структура проекта
```text
QR-Generator/
├── core/               # Ядро: логика генерации и обработки данных
│   ├── __init__.py
│   └── logic.py        # Основные функции (генерация, парсинг, очистка)
├── cli/                # Интерфейс командной строки
│   ├── __init__.py
│   └── main.py         # Точка входа для автоматизации
├── tests/              # Юнит-тесты для проверки стабильности
├── output/             # Результаты генерации (создается автоматически)
├── links.txt           # Входные данные (список ссылок)
├── Dockerfile          # Инструкции для сборки контейнера
├── .dockerignore       # Исключения для Docker
└── requirements.txt    # Зависимости проекта
```
## **Установка и запуск**
**Локально (через Python)**
+ Клонируйте репозиторий и создайте виртуальное окружение:
  python -m venv venv
  source venv/bin/activate  # Для Windows: venv\Scripts\activate

+ Установите зависимости:
  pip install -r requirements.txt
+ Запустите генерацию: Подготовьте файл links.txt и выполните:

  python cli/main.py

**Через Docker**
Проект полностью готов к работе в контейнере:
+ Сборка образа:

docker build -t qr-generator .

+ Запуск:

docker run --rm -v ${PWD}/output:/app/output qr-generator

## **Тестирование**
Для запуска автоматических тестов используйте:

python -m pytest tests/
## **Конфигурация**
+ Входной файл: links.txt (одна ссылка/текст на строку).
+ Папка вывода: output/QR_Results/.
+ Формат имен файлов: {номер}_{очищенная_ссылка}.png.
